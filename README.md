# Boiler Heat Flow Card

YAML-only Home Assistant custom card for thermal flows.

## Highlights
- Collector, boiler, hot water, fireplace, heat pump, floor and radiator nodes
- Animated moving dots on active flow lines
- YAML-only configuration
- Thresholds support both `temp` and `delta` mode per section
